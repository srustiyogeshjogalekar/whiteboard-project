Collaborative Real-Time Whiteboard


🚀 Overview

A real-time collaborative whiteboard application where multiple users can draw, chat, and collaborate together using shared rooms.

✨ Features
🎨 Real-time drawing (Canvas API)
👥 Multi-user collaboration (Socket.IO rooms)
🧽 Eraser tool & brush customization
🎨 Color picker
💬 Live chat system
🧹 Clear canvas sync
💾 Download board as image
🔗 Shareable room links
🛠️ Tech Stack

Frontend:

React.js
HTML5 Canvas
CSS

Backend:

Node.js
Express.js
Socket.IO

⚙️ Installation
Clone repo
git clone https://github.com/srustiyogeshjogalekar/whiteboard-project.git

Backend setup
cd server
npm install
npm start

Frontend setup
cd client
npm install
npm start

🌐 How it works
Create or join a room
Share link with others
Draw together in real-time
Chat and collaborate instantly
